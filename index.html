Get Started

INSTALLATION

Download, extract, run. No installer required.

01
Download the launcher
Click the download button and save the MODO-X-ADAPTED-Setup.exe file anywhere on your machine.
02
Run the installer
Double-click the setup file. Windows may show a SmartScreen warning — click More info → Run anyway.
03
Set your app path
On first launch, go to Settings → App Path and point it to your executable.
C:\Games\YourApp\YourApp.exe
04
Launch
Hit the orange Launch button. Customize your theme, enable particles, set your language — you're all set.
